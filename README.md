# 说明
> 这个项目用于写留言板实现的功能，顺带温习Vue的知识（后期会加上ElementUI）
> 如果对你有帮助，你可以点击右上角的"Star"支持一下 谢谢！
>  如有问题请直接在 Issues 中提，或者您发现问题并有非常好的解决方案，欢迎 PR 👍

## 项目运行（nodejs 6.0+）
``` bash
# 克隆到本地
git clone https://github.com/YingXiaoMing/VueMessageboard.git

# 进入文件夹
cd VueMessageboard

# 安装依赖
npm install

# 开启本地服务器localhost:8088
npm run serve

# 发布环境
npm run build
```
